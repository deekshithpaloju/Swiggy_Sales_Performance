# Swiggy_Sales_Performance
The Swiggy Sales Performance Dashboard project is a data analytics and visualization solution designed to provide actionable insights into food delivery operations using Swiggy-like transactional data. The primary objective of this project is to analyze sales trends, customer preferences, and restaurant performance to support data-driven decision.

📖 Overview

This project focuses on analyzing and visualizing sales data from a food delivery platform (Swiggy-like dataset). The dashboard provides insights into sales performance, customer behavior, and restaurant efficiency using interactive visualizations.

🎯 Objectives
Analyze overall sales performance
Identify top-performing cities and restaurants
Understand customer preferences and ordering patterns
Track key metrics like revenue, orders, and ratings
Enable data-driven decision-making
📂 Dataset Information

The dataset includes the following fields:

State & City
Order Date
Restaurant Name
Food Category / Item
Price (INR)
Ratings & Rating Count
🛠️ Tools & Technologies
Microsoft Excel (Pivot Tables, Charts, Dashboard) (or Power BI if you used it)
Data Cleaning & Transformation Techniques
Data Visualization
📊 Key Features of Dashboard
Total Revenue & Total Orders KPIs
Sales analysis by City and State
Top Restaurants by Revenue
Category-wise sales distribution
Time-based trends (monthly/daily)
Customer rating analysis
Interactive filters and slicers
📈 Insights Generated
Identification of high-demand cities
Popular food categories and items
Peak order periods
Correlation between ratings and sales
Performance comparison across restaurants
🚀 Conclusion

The dashboard provides a comprehensive view of sales performance and helps stakeholders optimize business strategies. It demonstrates the power of data analytics in improving operational efficiency and customer satisfaction in the food delivery industry.
